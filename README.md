# Tin-Dog-Project
<br>
TinDog is a mock dating website for dogs, inspired by the popular dating app Tinder. This project is built using HTML, CSS, and Bootstrap to practice front-end web development skills.

# Technologies Used<br>
- HTML5: For structuring the webpage.<br>
- CSS3: For styling the components.<br>
- Bootstrap 5: For responsive design and layout.<br>

# Features<br>
- Responsive layout for mobile, tablet, and desktop views.<br>
- Styled using Bootstrap components like cards, buttons, and grids.<br>
- Simple, modern, and clean UI.<br>
